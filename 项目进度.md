### 💭 关于出题机制的技术思考

你提的问题很好，我帮你分析两种方案的优劣：

| 方案 | 优点 | 缺点 | 适用场景 |
|-----|------|------|---------|
| **AI生成题目** | 题目无限、个性化强、可针对薄弱点 | 题目质量不稳定、可能超纲、格式不规范 | 日常练习、拓展训练 |
| **随机抽取题库** | 题目质量有保证、贴近真题、格式规范 | 题量有限、可能重复 | 模拟考试、真题训练 |
| **混合方案** | 结合两者优点 | 实现复杂 | **推荐！** |

**推荐方案：**
```
基础题库（真题+模拟卷）→ 随机抽取 → 保证质量
    ↓
薄弱知识点 → AI生成补充题 → 个性化训练
    ↓
新题加入题库 → 持续扩充
```

## 项目日志 - 2026-05-15 下午

### 数学模型测试结论

经过多轮测试（解三角形、函数集合、代数证明），得出以下结论：

| 模式 | 模型 | 平均耗时 | 适用场景 | 准确率 |
|-----|------|---------|---------|--------|
| ⚡ 快速模式 | qwen2-math:1.5b | 5-10秒 | 普通题型、日常练习、快速验证 | 中等 |
| 🚀 中速模式 | qwen2-math:7b | 20-30秒 | 疑难题目、代数证明、复杂计算 | 高 |
| 🎨 均衡模式 | qwen2.5-coder:7b | 40-60秒 | 需要美观公式、兼顾推理的场景 | 较高 |

### 最终配置

```javascript
const MODEL_CONFIGS = {
    math_fast: {
        model: 'qwen2-math:1.5b',
        name: '⚡ 快速模式',
        description: '速度最快(5-10秒)，适合普通题型、日常练习'
    },
    math_medium: {
        model: 'qwen2-math:7b',
        name: '🚀 中速模式',
        description: '速度中等(20-30秒)，适合疑难题目、代数证明'
    },
    math_balanced: {
        model: 'qwen2.5-coder:7b',
        name: '🎨 均衡模式',
        description: '速度较慢(40-60秒)，公式美观，兼顾推理'
    }
};
```
